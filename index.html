fiARworks

fiARworks is a gamified augmented reality app that allows the player to shoot fireworks that are overlayed on real live feed of camera video.

To build

git clone https://github.com/silverdev/fiARworks
$ cd fiARworks
$ echo "sdk.dir=<path to andoid sdk>" > local.properties 
$ ./gradlew installDebug

You need to have the android sdk and android build tools 18.1.0 installed
